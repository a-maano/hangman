## Description

The player would guess the letters of an unknown word before a figure of a hanged man is complete. 

## Getting Started

### Dependencies

* Made with Python 3.8.10 on Ubuntu 20.04.6 LTS 

### Installing

```
git clone https://github.com/a-maano/hangman.git
```

### Executing program

```
python3 hangman.py
```
